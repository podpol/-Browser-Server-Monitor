# 🚀 Browser Server Monitor

> Powerful Chrome extension for monitoring servers, viewing logs in real time, managing alerts, and analyzing issues with AI.

---

# ✨ Features

## 📡 Server Monitoring

- ✅ Monitor unlimited servers from one dashboard
- 🌐 HTTP and TCP (IP:PORT) ping support
- ⏱️ Custom ping interval (5 seconds – 1 hour)
- ⚠️ Custom latency warning threshold
- 📈 30-point real-time ping history graph
- 🟢🟡🔴 Online / Warning / Offline status indicators
- 🖼️ Custom server avatars or generated initials
- ↕️ Drag & Drop server reordering

---

## 📜 Log Viewer

- 🔄 Real-time log streaming
- 📂 Multiple logs per server
- 🚦 Automatic ERROR / WARNING / INFO detection
- 💬 Smart chat message parsing
- 🔍 Instant search with Regex support
- 🎯 Log level filters
- 📄 Word wrap toggle
- 💾 Download visible logs
- 📍 Smart auto-scroll
